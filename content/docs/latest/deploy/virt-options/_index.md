Flatcar Virtualization Options

The Flatcar Container Linux community has provided support for Flatcar Container Linux on a number of platforms beyond those [officially supported][official-support] (i.e, fully covered in the automated tests) by Kinvolk.

In addition to supported [Cloud Providers](../cloud/_index.md), Flatcar works the the community to support these other virtualization options:

## Other providers

- [Eucalyptus](./eucalyptus.md)
- [Vagrant](./vagrant.md)
- [VirtualBox](./virtualbox.md)
