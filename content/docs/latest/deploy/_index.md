# Flatcar Deployments

Flatcar provides three main deployment options:

- [Bare Metal](./bare-metal/_index.md) - Bare metal deployment of a Flatcar is the most popular and optimal experience.
- [Cloud Providers](./cloud/_index.md) - Provides easier deployment with the advantages with the cloud stack.
- [Virtualization options](./virt-options/_index.md) - Provides several tools for hosting Flatcar from other operating systems such as Windows and MacOS.
